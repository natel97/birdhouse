# Birdhouse

## About

Birdhouse is an API scaffolding tool for NestJS

## Usage

Usage: birdhouse `<command>`

where `<command>` is one of:
construct, init

## Example Syntax

```yaml
api:
  entities:
    user:
      primary-key:
        id: string

      schema:
        - name: string
        - email: string
        - age: number

      has-many:
        - article

    article:
      primary-key:
        id: string

      schema:
        - title: string
        - body: string
        - created: date
        - updated: date

      has-one:
        - user

      has-many:
        - tag

    tag:
      primary-key:
        name:
          type: string
          autogenerated: false

  routes:
    article:
      - INDEX
      - GET
      - POST
      - PUT
      - DELETE

    person:
      - INDEX
      - GET
      - POST
      - PUT
      - DELETE

    TAG:
      - INDEX
      - GET
```
